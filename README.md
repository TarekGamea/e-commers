# Store Engine Marketplace

A lightning-fast, full-stack e-commerce dashboard and storefront, stripped of heavy frontend framework overhead. This project utilizes a robust **Node.js/Express API ecosystem** on the backend paired with a high-performance **Vanilla JavaScript, Sass (SCSS), and Gulp-automated pipeline** on the frontend.

---

## 🗺️ Project Architecture

```text
ecommers-api/
├── backend/            # Express Application & Database Layer
│   ├── db/             # Mongoose Connection Drivers
│   ├── middleware/     # Global Interceptors & Error Handlers
│   ├── routes/         # Versioned REST Endpoint Hubs
│   ├── app.js          # Core Server Bootstrapping Script
│   └── seed.js         # Mock Database Seeding Script
└── frontend/           # Vanilla JS & SCSS Interface
    ├── js/             # Client-Side Logic Interceptors
    │   └── app.js      # Global Fetch Engine & DOM Pipeline
    ├── style/          # Modular SCSS Architectural Design Tokens
    ├── gulpfile.js     # Automation Compiler Definitions
    ├── index.html      # Application Shell Entry-point
    └── style.css       # Compiled Style Output
```
